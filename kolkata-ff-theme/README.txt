Kolkata FF Theme

Installation
1. Upload kolkata-ff-theme.zip in WordPress admin under Appearance > Themes > Add New > Upload Theme.
2. Activate "Kolkata FF Theme".
3. Open the new Kolkata FF menu in wp-admin.
4. Update branding, live results, tips, old results, hot numbers, page content, SEO text and contact details.
